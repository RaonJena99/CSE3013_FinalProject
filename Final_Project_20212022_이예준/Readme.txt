1. 실행 환경
프로그래밍 언어: C++
개발 환경: OpenFrameworks
테스트 환경: macOS Sonoma 14.3.1

2. 실행 방법
- 프로젝트 빌드 및 실행
  이 프로젝트는 OpenFrameworks를 사용하여 개발되었다. 
  따라서 OpenFrameworks의 설정 및 설치가 필요하다.

  - OpenFrameworks를 설치하고 프로젝트를 빌드하려면 다음 단계를 따르시오:
    OpenFrameworks를 공식 웹사이트에서 다운로드하고 설치한다.
    설치가 완료되면, 프로젝트 파일을 OpenFrameworks의 프로젝트 디렉토리에 복사한다.
    프로젝트를 열고 IDE에서 빌드 및 실행한다.

- 프로그램 사용 방법
  프로그램을 실행한 후 다음 키보드 명령을 사용할 수 있다:
  'l': 파일을 로드하여 경사면 정보를 불러온다.
  'd': 경사면 그리기 모드를 활성화한다.
  'e': 경사면 그리기 모드를 비활성화하고, 화면에 그려진 경사면을 삭제한다.
  'v': 현재 화면을 스크린샷으로 저장한다.
  'q': 프로그램을 종료한다.

3. 경사면 파일 형식
프로그램은 특정 형식의 텍스트 파일을 요구합니다. 
파일은 다음과 같은 형식으로 각 라인별로 저장되어야 한다:
------------------
number_of_lines
x1 y1 x2 y2
x1 y1 x2 y2
...
------------------
* number_of_lines: 경사면의 개수를 나타내는 정수.
* 각 x1, y1, x2, y2: 경사면의 끝점 좌표를 나타내는 정수.
< 주의!! > : 경사면은 서로 교차되어서는 안된다.
